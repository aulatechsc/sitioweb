Configurar acceso a GitHub

Para guardar cambios en el sitio necesitás un Personal Access Token de GitHub. Solo se pide una vez y queda guardado en este navegador.

  1. Entrá a github.com → Settings
  2. Ir a Developer settings → Personal access tokens → Tokens (classic)
  3. Clic en Generate new token (classic)
  4. Marcá el permiso repo
  5. Copiá el token y pegalo abajo
